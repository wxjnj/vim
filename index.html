gvim配置

  1. 安装gvim
  2. 复制.vim文件夹
  3. 复制.vimrc配置文件
  4. 安装ctags插件
  5. 安装窗口最大化插件wmctrl
  6. js语法检查插件jshint安装
    1. 安装node.js(需安装python2.6或2.7版本以及C++的编译器（GCC/G++))
      • wget http://nodejs.org/dist/v0.6.1/node-v0.6.1.tar.gz
      • tar zxvf node-v0.6.1.tar.gz
      • cd node-v0.6.1
      • ./configure
      • make
      • make install
    2. 安装npm
      • 在node文件夹下
      • curl http://npmjs.org/install.sh | sudo sh
    3. jshint安装
      • npm install jshint -g
